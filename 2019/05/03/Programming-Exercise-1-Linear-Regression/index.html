2 Linear regression with one variable

题目概述：采用线性回归的办法来预测餐车的利润，在题目中有城市的人口总量和利润数据。通过线性回归来选择在哪个城市放置餐车能使餐厅的利润最大化。

在开始执行任何任务之前，通过可视化来了解数据通常很有用。 对于此数据集，您可以使用散点图来显示数据，因为它只有两个要绘制的属性（利润和总体）。 （在现实生活中你会遇到的许多其他问题都是多维的，无法在二维图上绘制。）
