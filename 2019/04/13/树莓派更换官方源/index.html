树莓派更换官方源

树莓派3B+raspbin更新软件更换官方源

​ 第一次拿到树莓派，灰常激动，忙着开机，机器会弹窗提示你更新软件，结果更新了一下午，进度条才跑一半。原来是官方源国内访问过慢导致，下面更新一下软件源。身处国外忽略

strech说明现在是官方源。

下面需要修改/etc/apt下的sources.list文件

首先备份一下文件（不是必须

sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak

然后使用vi/vim/nano修改sources.list文件 注意：树莓派中vi/vim的操作极其反人类，建议先重装vim

sudo apt-get remove vim-commom //卸载vim

sudo apt-get install vim //安装新的vim

然后就可以以德芙丝滑般的手感将官方源注释掉并添加以下源：（没有打广告）

—阿里源

deb http://mirrors.aliyun.com/raspbian/raspbian/ stretch main non-free contrib

deb-src http://mirrors.aliyun.com/raspbian/raspbian/ stretch main non-free contrib

—中国科技大学源

deb http://mirrors.ustc.edu.cn/raspbian/raspbian/ stretch main contrib non-free rpi

接着你会发现etc/apt/sources.list.d下面还有个raspi.list文件里也有官方源

sudo vim /etc/apt/sources.list.d/raspi.list

接着注释掉然后添加源

最后！pi@raspberrypi:~*# sudo apt-get update

真香！！！！！
